# E's World Cup Pick'em 2026

## The World's Smallest Platform for the World's Largest Sporting Event
E's World Cup Pick'em 2026 is a full-stack web application that transforms a long-running family World Cup prediction tradition into an interactive online experience.

For decades, friends and family competed by predicting FIFA World Cup match results using handwritten scorecards and manually calculated standings. This project modernizes that experience by providing a centralized platform where users can submit predictions, manage their picks, and compete throughout the tournament.

### Features
#### Match Predictions
- Predict final scores for World Cup matches
- Save, edit, and delete predictions before kickoff
- Predictions automatically lock once a match begins

#### Real Match Data ⚽
- Displays official 2026 FIFA World Cup fixtures
- Match dates and tournament rounds included
- Team names displayed dynamically from the database

#### Prediction Locking 🔒 
- Users cannot modify predictions after kickoff
- Server-side validation prevents late changes

#### Full CRUD Functionality
- Create predictions
- Read saved predictions
- Update existing predictions
- Delete predictions before matches start

#### Competitive Gameplay 🎯
- Designed for friends, family, classrooms, and fan groups
- Foundation for automated scoring and leaderboards

#### How It Works
1) Browse upcoming World Cup matches.
2) Enter predicted scores for each fixture.
3) Save your prediction.
4) Edit or delete predictions before kickoff.
5) Once the match starts, predictions lock automatically.
6) Compare results and compete for bragging rights.
